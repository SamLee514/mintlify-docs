## Quick Example

```bash
# Install
curl -fsSL https://your-atlas-server/api/v1/cli/install.sh | bash

# Authenticate
atlas login

# Upload and generate report
atlas upload flight.bin --tailnum 436
```

That's it. Your Ghost Flight Report will be ready in minutes.

---

## What You'll Get

When you upload a `.bin` file, Atlas automatically:

<Steps>
  <Step title="Parses binary telemetry">
    Extracts all flight parameters from your bin file
  </Step>
  <Step title="Analyzes flight data">
    Runs AI-powered analysis on telemetry patterns
  </Step>
  <Step title="Generates insights">
    Creates a comprehensive Ghost Flight Report
  </Step>
  <Step title="Links to fleet">
    Associates the report with your aircraft asset
  </Step>
</Steps>

<Note>
  Add log files with `-l` for additional software and CAN bus analysis. The more data you provide, the deeper the insights.
</Note>
